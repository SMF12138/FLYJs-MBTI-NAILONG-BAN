# 角色人格测试 (MBTI-Character-Test)

一个基于15维度心理学理论的角色人格测试系统，通过选择题+隐藏彩蛋+速答的方式，测出用户的人格特征并映射到一个诙谐有趣的角色。

## 技术栈

- **前端**: Vue 3 + Vite + Pinia + Tailwind CSS
- **部署**: 静态文件（Nginx/CDN）
- **构建**: `npm run build` → `dist/`

## 快速启动

```bash
cd app
npm install
npm run dev      # 开发模式 http://localhost:5173
npm run build    # 生产构建
npm test         # 运行测试
```

## 15维度体系

| 维度 | 名称 | 说明 |
|------|------|------|
| D1 | 洞察 | 敏锐度、观察力 |
| D2 | 社交 | 外向性、社交偏好 |
| D3 | 谋断 | 决策速度、果断性 |
| D4 | 乐观 | 情绪稳定性、积极倾向 |
| D5 | 创造 | 开放性、创造力 |
| D6 | 自尊 | 自我价值感 |
| D7 | 共情 | 同理心、情感共鸣 |
| D8 | 信任 | 信任倾向、安全感 |
| D9 | 正义 | 道德判断、是非观 |
| D10 | 羞耻 | 道德敏感、羞耻感 |
| D11 | 勇敢 | 勇气、冒险倾向 |
| D12 | 恒心 | 坚持、延迟满足 |
| D13 | 洒脱 | 执念水平 |
| D14 | 贪婪 | 欲望驱动 |
| D15 | 幽默 | 幽默感 |

## 功能特性

- **27 道情境选择题**（含故事背景），15 维度人格测量
- **8 道限时速答**，超时扣分
- **隐藏彩蛋**：邀请函、1元测试、名字谜题、诗句拼图、广告关闭、血量清零特效等 11 个
- **结果页**：角色画像 + 人格解析 + 天作之合/夙世之敌 + 雷达图 + 匹配排行
- **血量系统**：3 滴血，扣血动画
- **结束流程**：隐藏关卡回顾 → 致谢 → 制作名单 → 参考文献 → 自动关闭
