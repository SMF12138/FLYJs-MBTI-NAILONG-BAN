# 角色人格测试 (MBTI-Character-Test)

> 一个基于15维度心理学理论的角色人格测试系统，通过情境选择题 + 隐藏彩蛋 + 限时速答，测出你的人格特征并映射到一个诙谐有趣的角色。

## 项目特色

- **15维度人格体系**：融合场依存理论、大五人格、共情理论等心理学模型，通过故事选择题全面测量人格特征
- **62个独特角色**：16正派 × 2反派 + 15高分特殊 + 15低分特殊，每个角色都有独立头像、诗句、前缀和5段式人格解析
- **沉浸式叙事**：贯穿全篇的"奶龙穿越"故事线，每个选择都影响剧情走向和角色判定
- **多维度检测机制**：正常题（27道）+ 限时速答（8道）+ 隐藏彩蛋（11种触发条件），全方位捕捉人格特征
- **天作之合与夙世之敌**：32个基础角色各有一对灵魂伴侣和宿敌，基于性格互补与价值观冲突设计

## 游戏流程

```
开始界面 → 前情提要 → 正常答题(27题) → 隐藏彩蛋(邀请函/广告/名字/诗句/救援)
   ↓
速答(8题) → 1元测试 → 结果页
   ↓
灰色地带选择(正/反) → 人格解析 → 基础画像匹配 → 特殊人格匹配
   ↓
天作之合 & 夙世之敌 → 维度雷达图 → 隐藏关卡回顾 → 结束流程
```

### 隐藏彩蛋触发条件

| 彩蛋 | 触发位置 | 触发条件 |
|------|---------|---------|
| 邀请函 | Q3之前 | 自动触发，展示隐藏条款 |
| 广告 | Q16选B后 | 展示30秒广告，可提前关闭 |
| 名字彩蛋 | Q17后 | 发现"奶龙"名字 |
| 谜题诗 | Q20后 | 拼出8句56字诗句 |
| 救援彩蛋 | Q21后 | 发现"王二狗"选项 |
| 血量清零 | HP为0时 | 特殊台词+界面特效 |
| 维度探索 | 答题阶段 | ↑↓键切换题目 |

### 血量系统

- 3滴血，部分选项扣血
- 扣血时：屏幕碎裂特效 + 冲击波 + 台词放大
- 清零时：特殊彩蛋"我奶龙，也绝不会倒下"
- 扣血台词轮换：强敌未收拾 → 罪孽未还清 → 错误未纠正

### 归一化算法

采用**动态极值归一化**，将所有得分来源（正常题 + 速答 + 隐藏彩蛋 + 1元测试）纳入理论极值计算：

1. 遍历所有题目，累加每个维度的理论最大/最小分
2. 叠加彩蛋/隐藏题的可能分数范围（邀请函、广告、名字彩蛋等）
3. 实际得分 = 题目分 + 彩蛋分，归一化为 `[(实际分 - 最小分) / (最大分 - 最小分) × 100]`
4. 未涉及维度 = 0

**心理理论支撑**：
- Jung, C.G. — 心理类型 (1921)
- Myers, I.B. & Briggs, K.C. — MBTI 指导手册
- McCrae & Costa — 大五人格理论 (1992)
- Bowlby, J. — 依恋理论 (1969)
- Kohlberg, L. — 道德发展阶段理论 (1981)
- Deci & Ryan — 自我决定理论 (1985)
- Batson, C.D. — 共情-利他假说 (1991)
- Aiken, L.R. — 耻感与罪感文化研究 (2002)

### 音效设计

- **全局按钮音效**：Web Audio API 合成 800→400Hz 正弦波（80ms）
- **扣血碎裂**：白噪音冲击（150ms衰减）
- **光波弹出**：锯齿波扫频 60→300→40Hz + 白噪音
- **文字放大**：正弦波深沉低频 120→30Hz（600ms）
