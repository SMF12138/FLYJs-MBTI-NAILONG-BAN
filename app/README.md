# MBTI-Character-Test

基于15维度心理学理论的角色人格测试系统。

## 开发

```bash
npm install
npm run dev      # 开发服务器 http://localhost:5173
npm run build    # 生产构建到 dist/
npm test         # 运行单元测试
npm run preview  # 预览生产构建
```

## 构建产物

构建后 `dist/` 目录为纯静态文件，可部署到任意静态文件服务器。

## 环境要求

- Node.js >= 18
- npm >= 9
