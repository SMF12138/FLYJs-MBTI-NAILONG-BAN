# 决策树：14维度全测方案

## 设计原则

1. **全维度覆盖**：每个维度都有独立题目，不跳过、不省略
2. **固定题量**：45-50题，用户一次性完成所有题目
3. **无分支跳转**：所有用户回答相同题组，难度一致
4. **分组排列**：按维度分组出题，避免连续同类题目造成疲劳
5. **题量分配**：每个维度 3-4 题，确保测量信度

每个维度得分范围 0-1，计算方式：

def calculate_dimension_scores(answers):
    """根据用户作答计算14维度得分"""
    scores = {}
    
    for dim in DIMENSIONS:
        dim_answers = [a for a in answers if a['dimension'] == dim]
        
        # 1. 题目答案得分（每题 0 或 1）
        answer_score = sum(a['score'] for a in dim_answers) / len(dim_answers)
        
        # 2. 行为数据修正（如果有）
        if dim in BEHAVIOR_DIMS:
            behavior_mod = calculate_behavior_modifier(dim, behavior_data)
            answer_score = answer_score * 0.7 + behavior_mod * 0.3
        
        scores[dim] = round(answer_score, 2)
    
    return scores

## 角色匹配算法

```python
import numpy as np

def match_character(user_scores, characters):
    """
    user_scores: 14维度得分字典
    characters: 角色库
    返回: 匹配度最高的角色
    """
    results = []
    
    for char in characters:
        # 提取角色向量和权重
        char_vector = np.array([char['traits'][f'D{i}_{name}'] 
                               for i, name in DIM_NAMES])
        weight_vector = np.array([char['weights'][f'D{i}_{name}'] 
                                  for i, name in DIM_NAMES])
        user_vector = np.array([user_scores[f'D{i}_{name}'] 
                                for i, name in DIM_NAMES])
        
        # 加权余弦相似度
        weighted_user = user_vector * weight_vector
        weighted_char = char_vector * weight_vector
        
        norm_user = np.linalg.norm(weighted_user)
        norm_char = np.linalg.norm(weighted_char)
        
        if norm_user == 0 or norm_char == 0:
            similarity = 0
        else:
            similarity = np.dot(weighted_user, weighted_char) / (norm_user * norm_char)
        
        results.append({
            'character': char,
            'similarity': similarity
        })
    
    # 按相似度排序
    results.sort(key=lambda x: x['similarity'], reverse=True)
    
    return results[0], results[:3]  # 最佳匹配 + Top3


# 维度名称映射
DIM_NAMES = [
    (1, 'insight'), (2, 'social'), (3, 'decision'), (4, 'emotion'),
    (5, 'explore'), (6, 'esteem'), (7, 'compassion'), (8, 'trust'),
    (9, 'moral'), (10, 'benevolence'), (11, 'courage'), (12, 'consistency'),
    (13, 'freedom'), (14, 'greed')
]
```

## 隐藏题：1元测试

正常48题结束后，显示感谢页面：

> **感谢参与！**
> 
> 本游戏正处于测试版，输入您的支付宝/银行卡号，点击确定即可获得 **1元** 奖励。

页面元素：确定按钮、取消按钮、银行卡号输入框（点击确定后出现）

### 行为流程

```
感谢页面（确定/取消）
  │
  ├── 点确定（记录点击速度）
  │     │
  │     ├── 迅速点确定 → D3谋断 +, D14贪舍 +
  │     ├── 慢速点确定 → D14贪舍 +, D3谋断 -
  │     └── 出现输入框（输入卡号并确定 / 未输入取消）
  │           │
  │           ├── 输入卡号并确定 → D8信疑 +, D14贪舍 +, D11勇懦 +
  │           └── 未输入取消 → D8信疑 -, D14贪舍 -, D11勇懦 -
  │
  └── 点取消/忽略（记录点击速度）
        │
        ├── 迅速点取消 → D3谋断 +, D14贪舍 -, D8信疑 -
        └── 慢速点取消/忽略 → D3谋断 -, D14贪舍 -, D8信疑 -
```

全程静默计分，不弹描述，结果统一在人物影像呈现。银行卡号不实际记录，仅检测"是否输入"行为。

3道陷阱题穿插在题目中，用于检测用户是否认真作答：

| 位置 | 陷阱题示例 | 正确答案 | 说明 |
|------|------------|----------|------|
| Q8 | "请选择非常同意" | 非常同意 | 注意力检测 |
| Q37 | "这道题没有正确答案，请选择B" | B | 指令遵循检测 |
| Q48 | "请选择非常不同意" | 非常不同意 | 一致性检测 |

**判定规则**：
- 3道陷阱题全部正确 → 信度高，完整输出结果
- 错1道 → 信度中等，结果标注"仅供参考"
- 错2道及以上 → 信度低，建议重新测试
