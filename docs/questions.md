# 题目体系设计

## 整体流程

```
开场介绍 + 血量系统说明（3滴血）
  │
  ▼
正常题目（25题，约10-12分钟）
  │
  ▼
隐藏题：1元测试
  │
  ▼
彩蛋题目（5个，穿插在正常题中）
  │
  ▼
隐藏速答环节（8题，5-60秒限时）
  │
  ▼
结果生成：14维度雷达图 + 角色匹配 + 分析文案
```

---

## 一、血量系统

- 开始时拥有3滴血
- 部分题目选择会扣血
- 扣血时界面变暗+阴影，弹出三次提示
- 血量清零后弹出彩蛋描述，血条恢复+颜色改变+界面高亮继续答题

**扣血触发条件**：
- Q1选择A（立刻前往）→ -1血
- Q25选择A（同归于尽）→ -1血
- 速答环节选A（杀尔伯）→ -1血

**扣血时提示文案**：
1. 我还有强敌未收拾！
2. 我还有罪孽未还清！
3. 我还有错误未纠正！

**血量清零彩蛋**：
> 我奶龙，也绝不能倒下！！！！！！！

```
  │     │
  │     ├── 迅速点确定 → D3谋断 +, D14贪舍 +

  │           │
  │           ├── 输入卡号并确定 → D8信疑 +, D14贪舍 +, D11勇懦 +
  │           └── 未输入取消 → D8信疑 -, D14贪舍 -, D11勇懦 -
  │
  └── 点取消/忽略（记录点击速度）
        │
        ├── 迅速点取消 → D3谋断 +, D14贪舍 -, D8信疑 -
        └── 慢速点取消/忽略 → D3谋断 -, D14贪舍 -, D8信疑 -
```
