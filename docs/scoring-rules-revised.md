# 评分规则与维度定义

> ⚠️ **重要说明**：下面的分值是经过修正的版本。请参照评分规则认真创作，所有分数都要基于题目，实际的合理的。

## 一、评分规则

### 1.1 分值范围
- 每道题每个维度的**基础分值**范围：**-1 到 1**
- 分值最多保留**两位小数**
- 最终得分 = 基础分值 × 权重

### 1.2 权重等级

| 权重 | 等级 | 适用场景 |
|------|------|----------|
| ×1 | 最低 | 题目与维度相关性弱，间接反映 |
| ×2 | 默认 | 题目能较好反映该维度（大多数题目） |
| ×3 | 较高 | 题目非常贴近生活，能真实反映人在该维度的表现 |
| ×4 | 高 | 题目是该维度的核心测量题，行为/选择高度对应维度特征 |
| ×5 | 最高 | 题目是该维度的"金标准"，几乎只能通过该维度解读 |

### 1.3 权重判定原则

**判断依据**：
1. **情境真实性**：题目情境是否贴近现实生活？越真实权重越高
2. **行为对应性**：选项行为是否直接对应维度特征？越直接权重越高
3. **心理可测性**：通过这个选择能否可靠推断出维度特征？越可靠权重越高
4. **维度纯净性**：这个选择是否只反映目标维度？混杂越少权重越高

### 1.4 基础分值判定原则

**判定步骤**：
1. 先确定该维度的**核心特征**（见下方维度定义）
2. 分析每个选项的行为/选择**指向哪个方向**
3. 根据指向强度给出-1到1的分值
4. 考虑**实际做题者心理**：人在真实情境下会怎么反应

**分值参考**：

| 分值 | 含义 | 行为特征 |
|------|------|----------|
| 1.0 | 完全符合维度A极 | 行为/选择完全指向A极 |
| 0.8 | 强烈符合A极 | 明显偏向A极，但有轻微犹豫 |
| 0.5 | 中等符合A极 | 偏向A极但程度中等 |
| 0.2 | 轻微符合A极 | 略微偏向A极 |
| 0 | 中性/不适用 | 不反映该维度或完全中立 |
| -0.2 | 轻微符合B极 | 略微偏向B极 |
| -0.5 | 中等符合B极 | 偏向B极但程度中等 |
| -0.8 | 强烈符合B极 | 明显偏向B极 |
| -1.0 | 完全符合B极 | 行为/选择完全指向B极 |

### 1.5 计算公式

```
单题单维度得分 = 基础分值 × 权重
某维度总得分 = 所有题目该维度得分之和
维度最终得分 = 某维度总得分 / 该维度出现次数（归一化）
```

## 二、十五维度定义

### D1 洞察（Insight）
**核心**：能否主动发现隐藏信息和环境异常

**A极（洞察者）特征**：
- 主动搜索隐藏元素
- 注意到常人忽略的细节
- 对异常敏感，直觉准

**B极（蒙昧者）特征**：
- 关注整体，忽略细节
- 对异常不敏感
- 不主动寻找隐藏信息

**测算角度**：
- 1：主动发现隐藏元素，无需提示
- 0：没有发现，也没有主动寻找
- -1：即使被提示也选择忽略

---

### D2 寡合（Social Energy）
**核心**：社交对自身是充电还是耗电

**A极（君羊者）特征**：
- 享受社交，人群中活力四射
- 主动寻求社交机会
- 独处时感到无聊

**B极（子瓜者）特征**：
- 社交消耗能量
- 偏好独处
- 需要安静空间恢复精力

**测算角度**：
- 1：主动选择社交场景，享受人群
- 0：被动参与，不排斥也不追求
- -1：主动回避社交，独处更舒适

---

### D3 谋断（Decision Style）
**核心**：决策时的犹豫程度和行动速度

**A极（决断者）特征**：
- 快速决策，定了就不回头
- 执行力强，"先干了再说"
- 不后悔已做的决定

**B极（拖延者）特征**：
- 反复权衡，选择困难
- "让我再考虑一下"
- 容易后悔已做的决定

**测算角度**：
- 1：几乎不思考直接行动
- 0：需要思考但最终能决定
- -1：反复权衡无法做出决定

---

### D4 悲欢（Emotional Tone）
**核心**：面对事件时的情绪基调和预期倾向

**A极（乐观者）特征**：
- 天生乐天，相信事情会变好
- 笑点低，正能量
- 将坏事归因为暂时的、特定的

**B极（悲观者）特征**：
- 谨慎预期，习惯想最坏情况
- 深沉内敛，危机意识强
- 将坏事归因为永久的、普遍的

**测算角度**：
- 1：自动往积极方向解读
- 0：中性看待
- -1：自动往消极方向解读

---

### D5 创造（Exploration Drive）
**核心**：对新事物的主动追求程度和创新倾向

**A极（探索者）特征**：
- 好奇心旺盛，喜欢尝试新事物
- 享受发现的过程
- 彩蛋猎人、创新先锋

**B极（守成者）特征**：
- 偏好稳定和熟悉
- 够用就行，不喜欢变化
- 实用主义

**测算角度**：
- 1：主动探索未知领域
- 0：满足于已知
- -1：抗拒任何新事物

---

### D6 荣辱（Honor）
**核心**：对外在评价的在意程度

**A极（狂傲者）特征**：
- 不在乎他人评价
- 我行我素，特立独行
- "别人怎么看我无所谓"

**B极（自卑者）特征**：
- 非常在意他人看法
- 怕丢面子，敏感于评价
- "不能让人看笑话"

**测算角度**：
- 1：完全无视他人评价
- 0：在意但不被左右
- -1：极度在意，活在他人眼光中

---

### D7 恻隐（Compassion）
**核心**：对他人苦难的感受力和帮助动机

**A极（共情者）特征**：
- 容易感同身受
- 见不得别人受苦
- "我看着都疼"

**B极（自我者）特征**：
- 理性旁观，边界感强
- 不容易被他人情绪影响
- "那是他的事"

**测算角度**：
- 1：强烈共情，愿意牺牲自己帮助他人
- 0：能理解但保持距离
- -1：完全无感，冷漠旁观

---

### D8 信疑（Trust）
**核心**：对他人/信息的信任倾向

**A极（宽容者）特征**：
- 信任他人，愿意给机会
- 包容缺点
- "他应该不是故意的"

**B极（谨慎者）特征**：
- 保持距离，观察后再信任
- 防备心强
- "先看看再说"

**测算角度**：
- 1：轻易相信，愿意接受解释
- 0：有条件信任，需要证据
- -1：高度怀疑，不接受任何解释

---

### D9 是非（Moral Judgment）
**核心**：对规则的坚守程度

**A极（守则者）特征**：
- 规则就是规则，黑白分明
- 原则性强
- "规定就是规定"

**B极（变通者）特征**：
- 具体情况具体分析
- 灵活处理，实用主义
- "这要看情况"

**测算角度**：
- 1：绝对遵守规则，不接受例外
- 0：有原则但能变通
- -1：完全无视规则，只看结果

---

### D10 羞恶（Moral Shame）
**核心**：对不道德行为的羞耻感

**A极（高洁者）特征**：
- 追求道德完美
- 对不洁事物强烈排斥
- "这事我做不出来"

**B极（无耻者）特征**：
- 不受道德束缚
- 实用主义至上
- "成王败寇"

**测算角度**：
- 1：对不道德行为有强烈排斥
- 0：能接受灰色地带
- -1：毫无羞耻感，成王败寇

---

### D11 勇懦（Courage）
**核心**：面对风险时的行动倾向

**A极（甬力者）特征**：
- 敢于冒险，迎难而上
- 行动力强
- "干就完了"

**B极（心去者）特征**：
- 谨慎保守，规避风险
- 三思后行
- "安全第一"

**测算角度**：
- 1：明知有风险仍然行动
- 0：评估风险后决定
- -1：极力规避风险，安全第一

---

### D12 恒变（Consistency）
**核心**：坚守与变通的倾向，逆境中的意志力

**A极（卫道者）特征**：
- 坚守原则，不喜欢变化
- 维护传统，稳定优先
- 逆境中能坚持不动摇

**B极（叛道者）特征**：
- 挑战权威，喜欢打破常规
- 追求革新，变化优先
- 逆境中容易放弃或转向

**测算角度**：
- 1：强烈偏好稳定，逆境中能坚持不动摇
- 0：能适应变化，逆境中能坚持但会调整策略
- -1：主动寻求变化，逆境中容易放弃或转向

---

### D13 逍遥（Freedom）
**核心**：对事物的执着/放下程度

**A极（通达者）特征**：
- 心态开放，不被外物束缚
- 随遇而安，内心平和
- "想开点就好"

**B极（羁绊者）特征**：
- 容易被牵绊，放不下过去
- 情感依赖
- "我放不下"

**测算角度**：
- 1：内心平静，能放下任何事物
- 0：有牵挂但不被束缚
- -1：严重依赖，无法放下

---

### D14 贪舍（Greed）
**核心**：对欲望的追求程度

**A极（贪婪者）特征**：
- 欲望强烈，追求更多
- 不容易满足
- "还要更多"

**B极（自由者）特征**：
- 欲望淡泊，容易满足
- 知足常乐
- "够用就行"

**测算角度**：
- 1：永不满足，贪得无厌
- 0：有欲望也懂节制
- -1：极低物欲，容易满足

---

### D15 幽默（Humor）
**核心**：面对困境时的调侃能力与幽默感

**A极（幽默者）特征**：
- 用幽默化解压力
- 自嘲能力强
- 能逗乐他人

**B极（严肃者）特征**：
- 严肃认真
- 不擅长也不喜欢幽默
- 对调侃不感冒

**测算角度**：
- 1：用幽默化解压力，自嘲能力强，能逗乐他人
- 0：有幽默感但不常用，偶尔会开玩笑
- -1：严肃认真，不擅长也不喜欢幽默

## 三、血量系统
- 开始时拥有3滴血
- 部分题目选择会扣血
- 扣血时界面变暗+阴影，弹出三次提示
- 血量清零后弹出彩蛋描述，血条恢复+颜色改变+界面高亮继续答题

**扣血触发条件**：
- Q1选择A（立刻前往）→ -1血
- Q25选择A（同归于尽）→ -1血
- 速答1选择A（杀尔伯）→ -1血

**扣血时提示文案**：
1. 我还有强敌未收拾！
2. 我还有罪孽未还清！
3. 我还有错误未纠正！

**血量清零彩蛋**：
> 我奶龙，也绝不能倒下！！！！！！！
